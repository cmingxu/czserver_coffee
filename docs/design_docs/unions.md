# API地址：



### `GET /unions/xxxxxx` 开启联盟
-

### `GET /unions/xxxxxx` 升级联盟等级（可以获得更多的捐塔空间）
-

### `GET /unions/xxxxxx` 获取联盟列表
-

### `GET /unions/xxxxxx` 加入联盟
-

### `GET /unions/xxxxxx` 联盟管理

- 创建联盟
- 批准是否加入
- 踢人

### `GET /unions/xxxxxx` 联盟聊天
- 获取聊天信息列表
- 发送捐塔请求
- 给他人捐塔
